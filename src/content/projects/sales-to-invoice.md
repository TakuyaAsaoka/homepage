実家には、どうしても紙が残る業務があります。手書きの売上明細を読み取り、そのまま請求書として印刷できるところまでを、ひとつのWebアプリにしました。

- **OCR** — Google Cloud Vision API で手書きの売上記録を読み取り、ステートマシンで請求書の項目を抽出する
- **PDF生成** — 使い慣れた既存の請求書用紙のフォーマットに合わせ、データ部分だけを座標指定で印字する（pdf-lib）
- **最小のUI** — ITに不慣れな利用者が、最小限の操作で完結できる画面設計
- **運用コスト** — GCPの無料枠に収まる構成（Cloud Run + Firestore）

多機能にするのではなく、「使う人がいちばん楽になる形」で確実に回り続けることを大事にしています。
